# MyPluginMaster

Custom Dalamud repository for bloooowfish plugins.

## Custom Repository URL

```text
https://raw.githubusercontent.com/bloooowfish/MyPluginMaster/refs/heads/main/repo.json
```

## Included Plugins

- Bazooka Lens
- Where Is My Head

## Maintenance

After plugin releases, run `tools\Update-MasterRepo.ps1 -Commit -Push` to refresh `repo.json`.
